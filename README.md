## Installation
**Set virtual environment:**
```bash
python3 -m venv .venv
```

**Activate virtual environment:**
- macOS/Linux
```bash
source .venv/bin/activate 
```  

- Windows:
```bash
.venv\Scripts\activate 
```  

**install dependencies:**
```bash
pip install -r requirements.txt 
```

## Run Flask
```bash
python ./app/app.py
```
